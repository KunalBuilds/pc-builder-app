# My PC App (Kotlin Android) - Step by Step Guide

## 1. Project Setup
- Create new Android project
- Language: Kotlin
- Min SDK: 21+

## 2. Features Overview
- Login (Google + Email)
- Create PC
- Compare PC
- Suggest upgrades
- Buy links (Amazon, Flipkart)
- Help section (Call + AI Chat)

## 3. Screens

### Screen 1: Login
- Google Sign-In
- Create Account

### Screen 2: Home
- Button 1: Let's Make New PC
- Button 2: Compare My PC

### Screen 3: Component Selection
- CPU
- GPU
- Motherboard
- RAM
- SSD
- Power Supply
- Cabinet
- Monitor

### Screen 4: Budget Input
- Enter budget

### Screen 5: Result
- Show selected PC
- Suggest better components in same budget
- Suggest missing upgrades

## 4. Backend
- Firebase Authentication
- Firebase Firestore (store user data)

## 5. APIs Needed
- Product API (for pricing)
- Affiliate links (Amazon/Flipkart)

## 6. Logic
- Compare price vs performance
- Suggest upgrades within same budget

## 7. Help Section
- Call button
- AI chatbot
